# 🎬 快速开始部署

## 方式一：使用自动化脚本（推荐）

在终端中运行：

```bash
cd /Users/lihenry/Documents/short_video
./deploy.sh
```

脚本会引导你完成整个部署过程。

## 方式二：手动部署

### 1. 创建 GitHub 仓库

访问：https://github.com/new

- Repository name: `short-video`
- 选择 **Public**
- **不要**勾选任何初始化选项
- 点击 **Create repository**

### 2. 推送代码

```bash
cd /Users/lihenry/Documents/short_video

# 添加远程仓库（替换 YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/short-video.git

# 推送代码
git branch -M main
git push -u origin main
```

### 3. 部署到 Vercel

1. 访问 https://vercel.com
2. 用 GitHub 登录
3. 点击 **"Add New"** → **"Project"**
4. 选择 `short-video` 仓库
5. 点击 **"Deploy"**
6. 等待 1-2 分钟
7. 获得访问链接！🎉

## 📱 测试你的网站

部署成功后，在手机上打开链接，测试：

- ✅ 上下滑动切换视频
- ✅ 顶部分类切换
- ✅ 点赞、评论、分享按钮
- ✅ 观看 5 个视频后弹出问卷
- ✅ 点击"组队探店"按钮

## 🔄 更新代码

修改后重新部署：

```bash
git add .
git commit -m "更新内容"
git push
```

Vercel 会自动重新部署！

## ⚠️ 注意事项

✅ **视频文件**：当前视频总大小约 18.9MB，可以直接推送到 GitHub

如果后续添加更大的视频（单个文件 >100MB），建议：
- 使用视频压缩工具
- 或使用云存储（阿里云 OSS、七牛云等）
